What is SkittleChat?

SkittleChat is a program for local communication with friends. It allows users to chat over a local network without the need for an internet connection.

Instruction for Server

Server version 1.0

If you want to use the server functionality, you need to configure message distribution via email. The host must enter their email credentials so the server can send notifications and manage connections properly.

Do not use the difrent version Server and SkittleChat!
