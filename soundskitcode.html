What is SoundSkiCode

Wait until program started!

This program allows you to encode and decode text by converting it into sound and back. Using a custom encoding method, each character of the text is transformed into a unique sound frequency, creating a sound file that represents the original text. The sound file can then be analyzed to decode and retrieve the original message.
