Čo je SkittleChat?

SkittleChat je program na lokálnu komunikáciu s priateľmi. Umožňuje používateľom komunikovať cez lokálnu sieť bez potreby pripojenia na internet.

Inštrukcie pre server

Server verzia 1.0

Aby ste mohli používať serverovú funkciu, musíte nakonfigurovať distribúciu správ cez e-mail. Hostiteľ musí zadať svoje prihlasovacie údaje, aby server mohol posielať upozornenia a správne spravovať pripojenia.

Nezabudnite používať rovnakú verziu servera a SkittleChat!
